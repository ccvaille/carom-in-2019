## 运行
```
npm i
gatsby develop
```

## 效果
使用 Videojs 库，定制播放器

## note
- 需要自定义字体图标可以参考 `https://github.com/videojs/font`，修改 `icons.json` 文件，需要指定 `root-dir`，`grunt` 打包后，复制 `woff` 字体到项目内即可；
- 现在的字体 `WOFF` 基本兼容所有了
